## Research Themes

### Multimodal Reasoning

I study how language, perception, and tool outputs can be fused for interpretable multi-step decisions.

The emphasis is on failure-aware reasoning, not only one-shot task accuracy.

### Agent Workflows

I build planner-executor-verifier pipelines that expose intermediate states and improve controllability.

This enables better auditability for long-horizon tasks with uncertain inputs.

### Edge AI

I explore deployment-aware design for constrained devices and GPU-limited settings.

Latency, memory, and robustness are treated as first-order research constraints.

### Scientific Discovery Systems

I investigate AI workflows for hypothesis generation, retrieval, and evidence-grounded synthesis.

The goal is reproducible, inspectable support for discovery-oriented decision-making.
